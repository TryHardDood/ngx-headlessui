Applied the entire time an element is entering. Usually you define your duration and what properties you want to transition here, for example transition-opacity duration-75. 
enter
Default value: 
none
The starting point to enter from, for example opacity-0 if something should fade in. 
enterFrom
Default value: 
none
The ending point to enter to, for example opacity-100 after fading in. 
enterTo
Default value:

Default value:
